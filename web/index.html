AeonShield

End-to-end pipeline for Business Email Compromise and phishing detection: Random Forest classification, SHAP explanations, stylometry, and optional organizational graph analysis.

Live dashboard: after you deploy the app on Streamlit Community Cloud, paste your app URL into web/index.html (see STREAMLIT_APP_URL below) and redeploy this site on Vercel.

Launch dashboard View on GitHub

Run locally: 

git clone https://github.com/sayuj5/AeonShield--Phishing-Intent-Analysis-Engine.git
cd AeonShield--Phishing-Intent-Analysis-Engine
python -m venv .venv
pip install -r requirements.txt
streamlit run dashboard.py
